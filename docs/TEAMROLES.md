# Team Roles

## RACI for Major Components

- **Backend API**: R: Backend Dev; A: Tech Lead; C: Security; I: Product
- **Agent Orchestration**: R: AI/ML Dev; A: Architect; C: Clinicians; I: Ops
- **UI**: R: Frontend Dev; A: UX Lead; C: Users; I: All
- **Infrastructure**: R: DevOps; A: CTO; C: Security; I: Team

## On-Call Expectations

- 24/7 rotation for P1 incidents
- Response: <15 min
- Handover: Daily standup

## PR Review Rules

- Required: 2 approvals, one from security for sensitive code
- Automated: Lint, tests, security scan
- Definition of Ready: Code complete, tests written, docs updated

## Agile Ceremonies

- Daily Standup: 15 min
- Sprint Planning: 2h
- Retro: 1h
- Demo: Bi-weekly

## Definition of Ready/Done

- Ready: AC defined, design approved
- Done: Code reviewed, tested, deployed, docs updated
